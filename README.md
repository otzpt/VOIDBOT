# VOIDBOT
A simple custom Slack bot created as my first learning project in bot development focuses on exploring APIs and server automation. 

I have no experience with Slack bots, webhooks, or server hosting, so this project is a total learning experience for me. During Stardance, I will learn how to set up a bot from scratch, connect it to the Slack API, and make it respond to three custom commands. I will also learn how to host it 24/7 so it stays online for the community to try out.
